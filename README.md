# 📦 Sistema de Almacén en Python

## 📌 Descripción

Este proyecto consiste en un sistema básico de almacén desarrollado en Python.
Permite agregar productos y visualizar la lista mediante un menú interactivo.

## 🎯 Objetivo

Aplicar los conceptos vistos en clase:
* Pilas
* Variables
* Listas
* Funciones (`def`)
* Ciclos `while` y `for`
* Condicionales `if`, `elif`, `else`

## 🛠️ Funcionalidades actuales

* Agregar productos
* Ver lista de productos
* Menú interactivo
* Salir del programa

## 📚 Tecnologías utilizadas

* Python

## ▶️ Cómo ejecutar

1. Descargar el archivo `almacen.py`
2. Abrir la terminal
3. Ejecutar el comando:

```
python almacen.py
```
